# Ace Theme
### Installation
1. Download from
2. Upload to Wordpress Installation
3. Install ACF Pro
4. Import file `acf-fields.json`

### Development
1. Fork the Repo
2. Run `npm install`
3. Commands
    - `gulp` - Watch styles and JavaScript and compile
    - `gulp javascript` - Compile just JavaScript
    - `gulp styles` - Compile just Styles
    - `gulp img` - Compile just Images

### How the Theme Works
